WordPress Installation Guide

Works perfectly with Elementor, Divi, Astra, GeneratePress, OceanWP, and all other themes. No plugin conflicts.

⚡ 5 Minutes🔌 No Plugin Needed✅ All Themes🌐 Global Sites
🎨

Any Theme

⚡

Instant Live

Plugin Option

Use "Insert Headers and Footers" plugin for easiest setup

1

Open Theme Editor

In WordPress admin, go to Appearance → Theme File Editor in the left sidebar.

2

Select footer.php

In the right panel list, find and click footer.php to open it in the code editor.

3

Find </body> Tag

Scroll to the bottom of the file. Find the closing </body> tag — usually the last few lines.

4

Paste Your Code

Paste your BotifyAI embed code directly above </body>. Then click Update File.

💡 Easier Method: Install the free "Insert Headers and Footers" plugin → paste code in Footer section. No file editing needed at all!
⚠️ If you update your theme, re-add the code. Use the plugin method to avoid this issue permanently.
