WordPress Installation

Works with any WordPress theme — Elementor, Divi, Astra, GeneratePress, and more.

1

Go to Appearance → Theme Editor

In your WordPress admin dashboard, navigate to Appearance → Theme File Editor (or Theme Editor).

2

Open footer.php

In the right panel, find and click on footer.php to open the file editor.

3

Paste before </body>

Find the closing </body> tag near the bottom. Paste your BotifyAI embed code just before it.

4

Click "Update File"

Save the file. Refresh your website — your AI bot will appear instantly in the bottom-right corner! 🎉

💡 Easier method: Install the free "Insert Headers and Footers" plugin. Then paste the embed code in the Footer section — no coding required.
🟩

Shopify Installation

Add BotifyAI to your Shopify store — boosts product discovery and customer support automatically.

1

Go to Online Store → Themes

In your Shopify admin, click Online Store in the left sidebar, then click Themes.

2

Click Actions → Edit Code

Find your current (live) theme. Click the Actions dropdown and select Edit code.

3

Open theme.liquid

In the Layout folder, click on theme.liquid to open it in the editor.

4

Paste before </body>

Scroll to the bottom, find the closing </body> tag, and paste your embed code just above it. Then click Save.

💡 Your AI chatbot will instantly appear on all pages of your Shopify store — product pages, checkout, homepage, and more.
🔵

Wix Installation

Add BotifyAI to your Wix website using the built-in Custom Code feature — no app needed.

1

Open Wix Editor → Settings

Go to your Wix dashboard. Click on Settings in the left menu, then select Custom Code.

2

Click "+ Add Custom Code"

In the Head or Body section, click the + Add Custom Code button.

3

Paste the Embed Code

Paste your BotifyAI embed code. Set placement to Body - end. Apply to All Pages.

4

Click Apply & Publish

Click Apply, then publish your site. Your AI bot will appear on all pages! 🎉

⚠️ Wix requires a paid plan to add custom code. If you're on a free Wix plan, upgrade first — or let our team handle it for you!
🟠

Webflow Installation

Perfect for Webflow sites — works with all templates and custom builds.

1

Open Project Settings

In the Webflow Designer, click the ⚙️ Settings icon (gear icon) at the top of the left panel.

2

Go to Custom Code Tab

Click the Custom Code tab in the Project Settings panel that opens.

3

Paste in "Footer Code" box

Find the Footer Code text area at the bottom. Paste your BotifyAI embed code there.

4

Save & Publish

Click Save Changes, then publish your site. The AI bot appears on your live site instantly!

💡 Webflow's Custom Code feature is available on all paid plans. Your bot will load on every page sitewide automatically.
🔧

Plain HTML / Custom Site

For custom-built websites, static HTML sites, or any platform not listed above.

1

Open your HTML file

Open your website's main HTML file (usually index.html) in any text editor — Notepad, VS Code, Sublime, etc.

2

Find the closing </body> tag

Scroll to the very end of the file. Look for </body> — it's almost always the second-to-last line.

3

Paste the embed code above it

Copy your BotifyAI embed code and paste it directly above the </body> tag.

4

Save and upload

Save the file and re-upload it to your hosting (cPanel, FTP, etc.). Your AI bot is now live! 🚀

💡 Need to add it to multiple pages? Paste the embed code in every HTML file you want the bot to appear on — or use a shared template/include file.
❓

Other Platforms

Using Squarespace, Blogger, Ghost, Joomla, or something else? We've got you covered.

1

Find "Custom Code" or "Embed" settings

Most platforms have a custom code section under Settings, Design, or Advanced options. Look for terms like Custom Code, Head/Footer Scripts, or Embed Code.

2

Paste in the Footer/Body section

Paste your BotifyAI embed code in the Footer or Body End section (not the Head). Apply to all pages.

3

Save and publish

Save your changes and publish/update your site. The AI chatbot will appear in the bottom-right corner on all pages.
