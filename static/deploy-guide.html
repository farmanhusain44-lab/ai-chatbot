WordPress

Most popular in UAE — used by 60% of businesses

1

Open WordPress Admin

Go to your website URL + /wp-admin
Example: mybusiness.ae/wp-admin
Login with your username and password.

2

Install Free Plugin (Easiest Way)

Left menu → Plugins → Add New
Search: Insert Headers and Footers
Click Install Now → then Activate

3

Paste the Code

Left menu → Settings → Insert Headers and Footers
Find the "Scripts in Footer" box
Paste the embed code there

4

Save & Done!

Click Save button → Visit your website → AI bot will appear in the bottom right corner ✅

💡 Alternative: Appearance → Theme Editor → footer.php → paste before </body> → Update File
🟩

Shopify

Popular for ecommerce stores in UAE & Saudi Arabia

1

Open Shopify Admin

Go to: yourstore.myshopify.com/admin
Login with your Shopify account.

2

Go to Theme Code

Left menu → Online Store → Themes
Click the "..." (3 dots) next to your current theme
Select "Edit Code"

3

Open theme.liquid

In the left file list, under Layout folder
Click on theme.liquid

4

Find </body> and Paste

Press Ctrl+F (or Cmd+F on Mac) → search </body>
Click just before </body> → paste the embed code

5

Save & Done!

Click Save → Visit your store → Bot appears bottom right ✅

Wix

Popular drag-and-drop website builder

1

Open Wix Dashboard

Go to: manage.wix.com → Login → Select your website

2

Go to Custom Code

Left menu → Settings → Custom Code
Click "+ Add Custom Code"

3

Paste & Configure

Paste the embed code in the box
Name: AI Chatbot
Add to pages: All Pages
Place code in: Body — end

4

Apply & Done!

Click Apply → Publish your site → Bot appears ✅

⚠️ Note: Wix Custom Code is only available on Premium plans. Free Wix accounts cannot add custom code.
⬜

Squarespace

Professional website builder — popular for portfolios & businesses

1

Open Squarespace Admin

Go to your Squarespace dashboard → Login

2

Go to Code Injection

Left menu → Settings → Advanced → Code Injection

3

Paste in Footer

Find the Footer text box → paste the embed code there

4

Save & Done!

Click Save → Visit your site → Bot appears bottom right ✅

Webflow

Used by designers and agencies in UAE

Paste in Footer Code

4

Publish & Done!

Custom / Developer Built

Website built by a developer using HTML/PHP/React etc.

1

Contact Your Developer

The fastest option — send your developer this message:

"Please add this script just before the </body> closing tag on all pages of our website. This will add a customer support chatbot."

[Then paste the embed code]
2

OR — Do It Yourself via cPanel

Login to your hosting cPanel → File Manager
Find public_html → open index.html or index.php
Click Edit → Find </body> → paste code before it → Save

3

Bot Live!

Visit your website → AI bot appears in bottom right corner ✅

💡 Common UAE hosting: SiteGround, Bluehost, HostGator, GoDaddy, Hostinger — all have cPanel File Manager

GoDaddy Website Builder

GoDaddy's own drag-and-drop builder

1

Login to GoDaddy

Go to: godaddy.com → My Products → Website Builder → Edit Website

2

Go to Website Settings

Click Settings (gear icon, top right) → Website Settings

3

Find Script Editor

Click "Edit HTML" or "Custom Scripts" tab
Paste the embed code in the Footer/Body section

4

Publish & Done!

Click Publish → Your bot is now live ✅

⚠️ If GoDaddy Builder doesn't allow custom scripts, use their WordPress hosting instead — it's more flexible.
