INDIA CUSTOMER SETUP

Launch Your BotifyAI
in Minutes

Follow these steps to add your multilingual AI customer-support chatbot to your Indian business website.

Step 1 — Copy your embed code

Your unique code is available in your dashboard after payment. Copy the complete line exactly as shown.

<script src="https://ai-chatbot-production-2f3a.up.railway.app/widget.js" data-agent="BotifyAI Assistant" data-access="YOUR-ACCESS-CODE"></script>

Step 2 — Add it to your website

1

WordPress

Install a header/footer plugin, open its Footer section, paste the code, and save.

2

Shopify

Go to Online Store → Themes → Edit Code → theme.liquid. Paste the code just before </body>.

3

Wix

Open Settings → Custom Code → Add Custom Code. Paste it on Body - end and apply it to all pages.

4

Custom HTML, Webflow or other platforms

Paste the code before the closing </body> tag, then publish your website.

Step 3 — Test your chatbot

Open your website in a new private/incognito window. Send a test message and confirm the BotifyAI chat button appears. The bot supports Hindi, English, Tamil, Telugu, Bengali, Marathi, Gujarati and 22+ Indian languages.
