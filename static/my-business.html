⚙️ Railway Variables (Email/WhatsApp Setup)

Yeh variables Railway pe set karne hain — railway.com → Project → Variables

# Gmail se auto email bhejne ke liye
GMAIL_USER = aapka@gmail.com
GMAIL_APP_PASSWORD = xxxx xxxx xxxx xxxx (App Password — Google Account se banao)

# Aapko notifications aane ke liye
OWNER_EMAIL = aapka@gmail.com
OWNER_WHATSAPP = +971xxxxxxxxx (aapka UAE number)

# Admin panel password change karna ho to
ADMIN_PASSWORD = farman2024
⚠️ Gmail App Password kaise banate hain:
1. myaccount.google.com kholo
2. Security → 2-Step Verification ON karo
3. Phir Security → App Passwords → "Mail" select karo → Generate
4. 16 digit code milega — woh Railway mein GMAIL_APP_PASSWORD mein paste karo
